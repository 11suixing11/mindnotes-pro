## 📋 两种部署方式

### 方式 1: GitHub Actions (推荐)

**步骤**:

1. 访问：https://github.com/11suixing11/mindnotes-pro/actions

2. 选择 **"Manual Deploy to GitHub Pages"**

3. 点击 **"Run workflow"**

4. 选择 **main** 分支

5. 点击 **"Run workflow"**

6. 等待 2-3 分钟

7. 访问：https://11suixing11.github.io/mindnotes-pro

---

### 方式 2: GitHub Pages 设置

**步骤**:

1. 访问：https://github.com/11suixing11/mindnotes-pro/settings/pages

2. **Source** 选择：**"Deploy from a branch"**

3. **Branch** 选择：**gh-pages**

4. **Folder** 选择：**/**

5. 点击 **Save**

6. 等待 2-5 分钟

7. 访问部署后的 URL
