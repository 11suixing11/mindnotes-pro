## 💡 关键决策

### 为什么选择 tldraw？

1. **功能完整** - 我们需要的都有
2. **维护活跃** - 持续更新
3. **文档完善** - 易于上手
4. **社区支持** - 问题有人帮
5. **可定制性** - 可以深度定制
6. **开源友好** - MIT 许可

### 为什么不是 excalidraw？

- tldraw 更轻量
- API 更现代化
- 定制更容易
- 更适合嵌入
