# Suggested Good First Issues

> 这些 issue 建议可以帮助新贡献者快速上手

## 🟢 Easy (适合新手)

### 1. 添加键盘快捷键提示
- **描述**: 在工具栏工具上添加 tooltip 显示键盘快捷键
- **标签**: `good first issue`, `enhancement`, `ui`
- **预计时间**: 2-3 小时
- **技能要求**: React, Tailwind CSS

### 2. 改进暗色模式对比度
- **描述**: 检查并修复暗色模式下的文字对比度问题
- **标签**: `good first issue`, `bug`, `accessibility`
- **预计时间**: 1-2 小时
- **技能要求**: CSS, 色彩理论

### 3. 添加更多导出格式
- **描述**: 支持导出为 SVG 格式
- **标签**: `good first issue`, `enhancement`, `export`
- **预计时间**: 3-4 小时
- **技能要求**: Canvas API, SVG

### 4. 添加撤销/重做快捷键提示
- **描述**: 在 UI 中显示 Ctrl+Z / Ctrl+Y 快捷键
- **标签**: `good first issue`, `enhancement`, `ux`
- **预计时间**: 1 小时
- **技能要求**: React

### 5. 改进文档文件夹排序
- **描述**: 支持按名称、日期、大小排序文档
- **标签**: `good first issue`, `enhancement`, `sidebar`
- **预计时间**: 2-3 小时
- **技能要求**: React, Zustand

- **预计时间**: 2 小时

- **描述**: 演示模式下的激光笔效果

---

## 如何创建这些 Issue

1. 在 GitHub 上创建新 issue
2. 使用对应的标签
3. 添加详细的描述和验收标准
4. 指定预计时间和技能要求
5. 欢迎新贡献者

## Issue 模板

```markdown
## 描述
[简要描述这个功能或修复]

## 动机
[为什么需要这个功能]

## 验收标准
- [ ] 标准 1
- [ ] 标准 2
- [ ] 标准 3

## 技术细节
[实现建议或参考]

## 预计时间
[X] 小时

## 技能要求
- React
- TypeScript
- [其他]
```
