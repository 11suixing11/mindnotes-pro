## ✨ What is MindNotes Pro?

**MindNotes Pro** is a natural handwriting note-taking tool that feels like pen and paper.

### 🎯 Quick Start

```bash
# Just open your browser and go to
https://mindnotes-pro.vercel.app
```

**That's it! No installation, no registration, no payment.**

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| ✍️ **Natural Writing** | Smooth strokes with perfect-freehand algorithm |
| 🎨 **Multi-Color** | Choose from preset colors or customize |
| 📏 **Adjustable Size** | 5 stroke thickness levels |
| ↩️ **Undo/Redo** | Never worry about mistakes |
| 💾 **Multi-Format Export** | PNG, PDF, or JSON for backup |
| ⌨️ **Keyboard Shortcuts** | Ctrl+Z (Undo), Ctrl+S (Save), Delete (Clear) |
| 🔒 **Privacy First** | All data stored locally in your browser |
| 📱 **Cross-Platform** | Works on any device with a browser |

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS 3
- **State Management**: Zustand
- **Drawing**: perfect-freehand
- **Export**: jsPDF, FileSaver.js

## 🚀 Getting Started

### Option 1: Use Online (Recommended)

```
1. Visit: https://mindnotes-pro.vercel.app
2. Start writing!
```

### Option 2: Self-Host

```bash
# Clone the repository
git clone https://github.com/11suixing11/mindnotes-pro.git
cd mindnotes-pro

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Option 3: Download Desktop App

Visit the [Releases](https://github.com/11suixing11/mindnotes-pro/releases) page to download:
- Windows (.msi)
- macOS (.dmg)
- Linux (.AppImage)

## 🎨 Usage

### Basic Operations

| Action | How To |
|--------|--------|
| **Write/Draw** | Click and drag with mouse |
| **Change Color** | Click color picker |
| **Adjust Size** | Click size selector |
| **Erase** | Click eraser tool |
| **Undo** | `Ctrl+Z` or click undo button |
| **Clear** | `Delete` or click clear button |
| **Save** | `Ctrl+S` or click save button |

### Export Options

- **PNG**: For sharing and presentations
- **PDF**: For printing and formal documents
- **JSON**: For backup and re-editing
