# 🚀 使用 Vercel 部署（超简单！）

## 为什么选择 Vercel

- ✅ **无需配置** - 自动检测 Vite
- ✅ **完全免费** - 个人版足够用
- ✅ **自动 HTTPS** - 自动配置 SSL
- ✅ **全球 CDN** - 访问速度快
- ✅ **自动部署** - push 代码自动更新

---

## 📋 部署步骤（只需 3 分钟）

### 步骤 1: 访问 Vercel

打开：https://vercel.com/new

### 步骤 2: 登录

使用 GitHub 账号登录（推荐）

### 步骤 3: 导入项目

1. 点击 **"Import Git Repository"**
2. 找到 **mindnotes-pro**
3. 点击 **"Import"**

### 步骤 4: 配置（无需修改）

- Framework Preset: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

### 步骤 5: 点击 Deploy

等待 1-2 分钟

### 步骤 6: 完成！

获得你的专属域名：
```
https://mindnotes-pro-xxx.vercel.app
```
