## 🚀 制作步骤

### 步骤 1：准备应用文件

```bash
# 复制项目文件到发布目录
cp -r mindnotes-pro/ MindNotes-Pro/

# 添加启动脚本（已创建）
# start.bat 已在项目中
```

### 步骤 2：下载 Node.js 安装包

访问：https://nodejs.org/
下载：`node-v20.x.x-x64.msi`（LTS 版本）

放到发布包文件夹。

### 步骤 3：创建压缩包

**方式一：使用 Windows 右键**

1. 选中所有文件
2. 右键 → 发送到 → 压缩文件夹
3. 重命名为：`MindNotes-Pro-v1.0.0-Windows.zip`

**方式二：使用 7-Zip**

1. 安装 7-Zip
2. 右键 → 7-Zip → 添加到压缩包
3. 选择压缩格式（ZIP/7Z）
4. 点击确定

## 📋 用户安装流程

### 首次使用（需要安装 Node.js）

```
1. 下载并解压发布包
2. 运行：nodejs-installer.exe
3. 安装 Node.js（一直点下一步）
4. 重启电脑
5. 双击：启动应用.bat
6. 浏览器自动打开
7. 开始使用！
```

**预计时间**: 5 分钟

## 💡 进阶优化

### 1. 创建安装包（.exe）

使用工具：
- **Inno Setup** - 免费
- **NSIS** - 免费
- **InstallShield** - 付费

功能：
- 一键安装
- 自动检测依赖
- 创建桌面快捷方式
- 卸载程序

### 2. 使用 Electron 打包

将 Web 应用打包成桌面应用：

```bash
npm install electron electron-packager -D
npm run build:win
```

输出：
- `.exe` 可执行文件
- 无需安装 Node.js
- 独立运行

### 3. 使用 PWA

添加 PWA 支持：

```
- 可安装到桌面
- 离线使用
- 类似原生应用
```
