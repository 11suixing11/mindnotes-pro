### 新增组件

#### Motion 动画组件

```typescript
// 淡入效果
<FadeIn delay={0.2}>
  <div>内容</div>
</FadeIn>

// 滑入效果
<SlideIn direction="up">
  <div>内容</div>
</SlideIn>

// 缩放效果
<ScaleIn>
  <div>内容</div>
</ScaleIn>

// 交错容器
<StaggerContainer staggerDelay={0.1}>
  {items.map(item => <FadeIn>{item}</FadeIn>)}
</StaggerContainer>
```

#### VirtualList 虚拟列表

```typescript
<VirtualList
  items={layers}
  itemHeight={50}
  overscan={5}
  renderItem={(layer, index) => (
    <LayerItem key={layer.id} layer={layer} />
  )}
/>
```
