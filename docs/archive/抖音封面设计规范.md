## 🎯 封面设计原则

### 3 秒法则

```
用户刷抖音时，每个视频只有 3 秒吸引注意力

封面必须做到:
✅ 一眼看懂（主题清晰）
✅ 视觉冲击（对比强烈）
✅ 情绪共鸣（痛点/收益）
✅ 引导点击（好奇心）
```

### 设计公式

```
好封面 = 大字标题 + 高对比配色 + 产品截图 + 情绪表达
```

## 📝 封面模板（10 款）

### 模板 1：对比型

**适用**: 对比测评类视频（视频 4、10）

```
┌─────────────────────┐
│                     │
│   [产品 A]  VS  [产品 B]  │
│      Logo    Logo   │
│                     │
│   ┌──┐      ┌──┐   │
│   │截│      │截│   │
│   │图│      │图│   │
│   │1│      │2│   │
│   └──┘      └──┘   │
│                     │
│   哪个更好用？      │
│                     │
└─────────────────────┘
```

**设计要点**:
- 左右分屏，对比强烈
- 中间"VS"大字
- 底部引导问题

**配色方案**:
- 左：蓝色系 (#1E90FF)
- 右：橙色系 (#FF8C00)
- 文字：白色 + 黑色描边

## 💡 封面优化技巧

### 提升点击率

1. **数字吸引** - "5 个功能"比"功能介绍"点击率高 30%
2. **对比强烈** - "vs"类封面点击率高 25%
3. **疑问句** - "怎么选？"比"选择指南"点击率高 20%
4. **收益明确** - "效率翻倍"比"效率提升"点击率高 15%
5. **紧迫感** - "要火"比"推荐"点击率高 35%

### 避免踩坑

- ❌ 文字太多（看不清）
- ❌ 配色太花（视觉混乱）
- ❌ 主题不明（不知道讲什么）
- ❌ 字体太小（手机上看不清）
- ❌ 与内容不符（点击后流失）

## 📊 封面数据追踪

### 核心指标

| 指标 | 计算方式 | 健康值 |
|------|---------|--------|
| **点击率** | 播放量/曝光量 | >10% 合格，>20% 优秀 |
| **完播率** | 完播/播放 | >30% 合格，>50% 优秀 |
| **转粉率** | 新增关注/播放 | >1% 合格，>2% 优秀 |

### 优化循环

```
发布 → 数据追踪 → 分析原因 → 优化封面 → 再次发布
```
