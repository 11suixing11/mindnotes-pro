## [1.1.0] - 2026-03-19 - 深色模式 🌙

### ✨ 新增功能

#### 深色模式 ⭐⭐⭐⭐⭐

**描述**: 完整的深色主题支持，保护眼睛，提升夜间使用体验

**功能特点**:
- ✅ 一键切换深色/浅色模式
- ✅ 自动检测系统主题偏好
- ✅ 记忆用户偏好设置
- ✅ 平滑过渡动画
- ✅ 全局主题一致性

**技术实现**:
- CSS 变量主题系统
- Zustand 状态管理
- localStorage 持久化
- prefers-color-scheme 媒体查询

**使用方法**:
1. 点击工具栏右上角的 🌙/☀️ 按钮
2. 或等待系统自动切换

**文件变更**:
- 新增：`src/store/useThemeStore.ts` - 主题状态管理
- 修改：`src/index.css` - CSS 变量定义
- 修改：`src/components/Toolbar.tsx` - 添加主题切换按钮
- 修改：`src/App.tsx` - 主题初始化

---

### 🎨 设计细节

#### 颜色方案

**浅色主题**:
```css
--bg-primary: #ffffff
--bg-secondary: #f9fafb
--text-primary: #1f2937
--text-secondary: #6b7280
--border-color: #e5e7eb
--primary-color: #4f46e5
```

**深色主题**:
```css
--bg-primary: #1f2937
--bg-secondary: #111827
--text-primary: #f9fafb
--text-secondary: #9ca3af
--border-color: #374151
--primary-color: #6366f1
```

### 📝 使用说明

#### 手动切换

1. 点击工具栏右上角的 🌙 或 ☀️ 按钮
2. 主题立即切换
3. 偏好自动保存

#### 自动切换

1. 首次访问时自动检测系统主题
2. 系统主题变化时自动跟随（如果未手动设置）
3. 手动设置后不再自动跟随
