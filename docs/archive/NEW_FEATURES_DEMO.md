## ⚡ 命令面板 (Command Palette)

### 如何打开

**快捷键**: `Ctrl+P` (Windows/Linux) 或 `Cmd+P` (Mac)

### 功能演示

```
┌─────────────────────────────────────────────────┐
│ ⚡ 输入命令或搜索功能... (按 Esc 关闭)           │
├─────────────────────────────────────────────────┤
│ ⚡ 操作                                          │
│   📝 新建笔记                    Ctrl+N         │
│   💾 保存笔记                    Ctrl+S         │
│   🗑️ 清空画布                                  │
│                                                   │
│ ➕ 插入                                          │
│   📝 插入文本框                   T             │
│   ⬜ 插入形状                     R             │
│   🖼️ 插入图片                   Ctrl+I         │
│   📋 使用模板                                   │
│                                                   │
│ 👁️ 视图                                          │
│   🎨 切换主题                   Ctrl+Shift+L   │
│   📐 切换网格                                   │
│   🔍 放大                       Ctrl++         │
│   🔍 缩小                       Ctrl+-         │
│                                                   │
│ 📤 导出                                          │
│   📤 导出为图片                                 │
│   📄 导出为 PDF                                 │
│   📝 导出为 Markdown                            │
└─────────────────────────────────────────────────┘
```

### 使用技巧

1. **快速搜索**: 输入关键词过滤命令
   - 输入 "保存" → 显示保存相关命令
   - 输入 "export" → 显示导出相关命令

2. **键盘导航**:
   - `↑` `↓` - 上下选择
   - `Enter` - 执行选中命令
   - `Esc` - 关闭面板

3. **类别分组**: 命令按功能分类，一目了然

## 📋 智能模板系统

### 如何打开

**方式 1**: 命令面板 → 输入 "模板" → 选择 "使用模板"  
**方式 2**: 工具栏 → 模板按钮 (开发中)

### 可用模板

#### 1. 📝 会议纪要

**适用场景**: 团队会议、客户会议、项目讨论

**包含内容**:
- 会议基本信息（日期、时间、地点、参会人）
- 会议议题
- 讨论要点
- 决策事项
- 待办行动（含负责人和截止日期）
- 下次会议安排

**预览**:
```markdown
# 会议纪要

**日期**: _________
**时间**: _________
**地点**: _________
**参会人**: _________

---

## 会议议题

1. _______________
2. _______________

---

## 决策事项

- [ ] 
- [ ] 

---

## 待办行动

| 事项 | 负责人 | 截止日期 |
|------|--------|----------|
|      |        |          |
```

---

#### 2. 📚 学习笔记

**适用场景**: 课堂笔记、在线课程、读书笔记

**包含内容**:
- 主题和来源
- 核心概念
- 重点总结
- 疑问与思考
- 复习计划

**特色**: 康奈尔笔记法结构

---

#### 3. 💡 头脑风暴

**适用场景**: 创意发想、问题解决、策划方案

**包含内容**:
- 核心问题
- 想法收集区
- 最佳创意评选
- 下一步行动计划

**特色**: 自由发散的结构

---

#### 4. 📋 待办事项

**适用场景**: 每日计划、任务管理

**包含内容**:
- 今日必做 (Top 3)
- 其他任务清单
- 时间安排表
- 今日完成
- 明日计划

**特色**: 优先级管理

---

#### 5. 🎯 目标规划

**适用场景**: 年度目标、项目规划、个人发展

**包含内容**:
- SMART 分析
- 行动计划（分阶段）
- 进度追踪表
- 激励语

**特色**: SMART 原则指导

---

#### 6. 📔 每日笔记

**适用场景**: 日记、每日复盘

**包含内容**:
- 晨间日记（目标、感恩）
- 晚间复盘（完成、收获、反思）
- 灵感记录
- 今日学习

**特色**: 完整的每日流程

## 💡 使用场景

### 场景 1: 突然的会议

```
1. 打开 MindNotes Pro
2. Ctrl+P 打开命令面板
3. 输入 "模板"
4. 选择 "会议纪要"
5. 开始记录！
```

**时间节省**: 30 秒 vs 5 分钟（手动创建结构）

---

### 场景 2: 学习新知识

```
1. 打开命令面板
2. 选择 "学习笔记" 模板
3. 按照康奈尔结构记录
4. 自动包含复习计划
```

**效果**: 结构化笔记，复习更高效

---

### 场景 3: 项目管理

```
1. 使用 "目标规划" 模板
2. SMART 分析目标
3. 分阶段行动计划
4. 进度追踪
```

**效果**: 目标清晰，执行有力
