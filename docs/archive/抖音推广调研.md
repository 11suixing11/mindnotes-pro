## 🎯 抖音推广策略

### 内容定位

**账号人设**:
```
名称建议:
- MindNotes 官方
- 小雨的笔记神器
- 灵感记录官

人设定位:
- 专业：笔记/效率工具专家
- 亲和：像朋友一样分享
- 实用：每条视频都有收获
```

**内容方向**（优先级排序）:

#### P0 - 核心内容（占比 50%）

**使用场景展示**:
```
选题示例:
1. 「考研党这样记笔记，效率翻倍」
2. 「会议记录神器，老板都夸我」
3. 「设计师的灵感收集本」
4. 「程序员的技術笔记管理」
5. 「学生党必备免费笔记 APP」

视频结构:
0-3s: 痛点引入（"还在为笔记混乱烦恼？"）
3-15s: 场景展示（手写/文字切换）
15-30s: 核心功能演示（模板/导出）
30-45s: 效果对比（使用前 vs 使用后）
45-60s: 引导行动（"评论区获取链接"）
```

#### P1 - 干货内容（占比 30%）

**方法论 + 工具结合**:
```
选题示例:
1. 「康奈尔笔记法实战演示」
2. 「费曼技巧用这个 APP 超好用」
3. 「如何建立个人知识体系」
4. 「5 步做出高颜值笔记」
5. 「手写 vs 键盘，哪个更利于记忆？」

视频特点:
- 有理论支撑（建立专业度）
- 有实操演示（展示产品功能）
- 有收获感（提升完播率）
```

#### P2 - 对比/测评内容（占比 20%）

**横向对比**:
```
选题示例:
1. 「5 款免费笔记 APP 横评」
2. 「MindNotes vs GoodNotes，怎么选？」
3. 「开源笔记应用哪家强？」
4. 「学生党笔记 APP 推荐 Top5」
5. 「这款开源 APP 要火」

视频特点:
- 客观公正（不拉踩）
- 突出差异化（手写 + 文字）
- 引导讨论（评论区互动）
```

### 视频制作规范

#### 技术参数

| 项目 | 要求 | 说明 |
|------|------|------|
| **分辨率** | 1080x1920 | 竖屏 9:16 |
| **时长** | 30-60 秒 | 完播率优先 |
| **格式** | MP4 | H.264 编码 |
| **大小** | <50MB | 上传限制 |
| **帧率** | 30fps | 流畅度 |
| **BGM** | 抖音热门音乐 | 提升推荐 |

#### 内容结构

```
黄金 3 秒法则:
- 第 1 秒：视觉冲击（高颜值笔记特写）
- 第 2 秒：痛点共鸣（"你是不是也...?"）
- 第 3 秒：价值承诺（"这个 APP 帮你解决"）

中间 30 秒:
- 功能演示（录屏 + 解说）
- 场景展示（真实使用）
- 效果对比（前后差异）

最后 10 秒:
- 总结价值（"免费开源，快去试试"）
- 引导行动（"评论区见"）
- 引导关注（"下期分享 XX 技巧"）
```

#### 封面设计

```
要素:
- 大字标题（15-20 字以内）
- 高对比度配色
- 产品界面截图
- 情绪表达（惊讶/满意）

标题公式:
- 数字 + 效果：「5 个功能让你效率翻倍」
- 痛点 + 解决：「笔记混乱？这个 APP 救了我」
- 对比 + 悬念：「比 GoodNotes 还好用？」
- 人群 + 场景：「考研党必备笔记神器」
```

## 💡 关键建议

### 必须做好的

1. **内容质量** — 抖音是内容驱动，内容不好投放也白搭
2. **前 3 秒** — 决定用户是否划走，必须抓眼球
3. **持续输出** — 算法喜欢稳定更新的账号
4. **互动运营** — 评论区及时回复，提升活跃度
5. **数据复盘** — 每周分析数据，优化内容方向

### 需要避免的

1. ❌ 硬广思维 — 抖音用户反感硬广
2. ❌ 断更 — 断更会掉权重
3. ❌ 刷数据 — 会被限流
4. ❌ 违规内容 — 注意平台规则
5. ❌ 急于求成 — 需要耐心积累

## 🎯 总结

### 抖音渠道优势

✅ **用户基数大** — 6 亿日活，流量池充足  
✅ **目标用户重合** — 18-35 岁占比高  
✅ **内容形式适合** — 视频可直观展示产品  
✅ **算法公平** — 新号也有机会爆  
✅ **转化路径短** — 主页链接直达  

### 抖音渠道挑战

❌ **竞争激烈** — 同类内容多  
❌ **内容要求高** — 需要持续产出优质内容  
❌ **投放成本上涨** — DOU+ 价格逐年上升  
❌ **用户耐心低** — 3 秒不吸引人就划走  

### 建议

**立即行动**:
1. 创建抖音企业号
2. 准备 5-10 条视频素材
3. 制定内容日历
4. 设置投放预算
5. 开始发布 + 测试

**长期心态**:
- 抖音是长期渠道，不是速成
- 前 3 个月可能效果一般，需要坚持
- 一旦打造出爆款，增长会加速
- 内容质量是核心，投放是放大器
