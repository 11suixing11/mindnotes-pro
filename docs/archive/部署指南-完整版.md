## 1. 在线部署（推荐）⭐⭐⭐⭐⭐

### 1.1 Vercel 部署（5 分钟）

#### 前置条件

- ✅ GitHub 账号
- ✅ 代码已推送到 GitHub

#### 部署步骤

**步骤 1**: 访问 Vercel

```
https://vercel.com/new
```

**步骤 2**: 导入项目

1. 点击 "Import Git Repository"
2. 搜索 `mindnotes-pro`
3. 点击 "Import"

**步骤 3**: 配置构建

```
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
```

**步骤 4**: 部署

点击 "Deploy"，等待 1-2 分钟

#### 部署成功

```
✅ 获得网址：https://mindnotes-pro.vercel.app
✅ 自动 HTTPS
✅ 全球 CDN
✅ 自动部署（每次 push 自动更新）
```

## 2. 桌面应用打包

### 2.1 Tauri 打包（推荐）⭐⭐⭐⭐⭐

#### 安装依赖

```bash
# 安装 Tauri CLI
npm install -D @tauri-apps/cli

# 初始化 Tauri
npx tauri init
```

#### 配置 tauri.conf.json

```json
{
  "build": {
    "beforeBuildCommand": "npm run build",
    "beforeDevCommand": "npm run dev",
    "devPath": "http://localhost:3000",
    "distDir": "../dist"
  },
  "package": {
    "productName": "MindNotes Pro",
    "version": "1.0.0"
  },
  "tauri": {
    "bundle": {
      "active": true,
      "targets": ["msi", "app"],
      "identifier": "com.mindnotes.pro",
      "icon": ["icons/icon.ico"]
    }
  }
}
```

#### 打包

```bash
# Windows
npm run tauri build

# 输出：src-tauri/target/release/bundle/msi/MindNotes Pro_1.0.0_x64_en-US.msi
```

#### 测试安装包

1. 下载 .msi 文件
2. 双击安装
3. 桌面会出现快捷方式
4. 双击启动

---

### 2.2 Electron 打包（备选）

#### 安装依赖

```bash
npm install -D electron electron-builder
```

#### 配置 electron-builder.yml

```yaml
appId: com.mindnotes.pro
productName: MindNotes Pro
directories:
  output: release
files:
  - dist/**/*
  - package.json
win:
  target:
    - nsis
    - portable
```

#### 打包

```bash
npm run build
npx electron-builder --win
```

## 3. 移动端打包

### 3.1 Capacitor + Android

#### 安装依赖

```bash
npm install @capacitor/core @capacitor/cli
npx cap init
```

#### 配置 capacitor.config.ts

```typescript
export default {
  appId: 'com.mindnotes.pro',
  appName: 'MindNotes Pro',
  webDir: 'dist',
  server: {
    androidScheme: 'https'
  }
}
```

#### 添加 Android 平台

```bash
npx cap add android
```

#### 构建

```bash
# 构建 Web 项目
npm run build

# 同步到 Android
npx cap sync

# 打开 Android Studio
npx cap open android
```

#### 生成 APK

在 Android Studio 中：

1. Build → Generate Signed Bundle / APK
2. 选择 APK
3. 创建或选择密钥
4. 选择 release 构建类型
5. 完成

**输出**: `app/release/app-release.apk`

---

### 3.2 Capacitor + iOS

#### 添加 iOS 平台

```bash
npx cap add ios
```

#### 构建

```bash
npm run build
npx cap sync
npx cap open ios
```

#### 打包

在 Xcode 中：

1. 选择 "Any iOS Device"
2. Product → Archive
3. 导出 IPA
