- [ ] 安装 OBS Studio（或其他录制软件）
- [ ] 安装 DaVinci Resolve（或 CapCut）
- [ ] 安装 Audacity（录制旁白）
- [ ] 准备麦克风（如有旁白）
- [ ] 清理电脑桌面
- [ ] 关闭无关应用和通知

### 2. 项目准备

- [ ] 确保项目可以正常运行
- [ ] 测试所有功能正常
- [ ] 准备演示用的示例内容
- [ ] 调整浏览器窗口大小（1920x1080）
- [ ] 清除浏览器缓存和历史

### 3. 脚本准备

- [ ] 阅读并熟悉演示视频脚本
- [ ] 标记重点演示步骤
- [ ] 准备旁白稿（如需旁白）
- [ ] 准备背景音乐
- [ ] 准备音效文件

## 🎬 录制阶段

### 4. 录制设置

- [ ] 设置录制分辨率：1920x1080
- [ ] 设置帧率：60fps
- [ ] 设置码率：10-20 Mbps
- [ ] 设置录制区域：浏览器窗口
- [ ] 测试音频录制（如有旁白）
- [ ] 测试鼠标高亮（可选）

### 5. 录制演示

#### 开场（10 秒）

- [ ] 展示完整界面
- [ ] 保持画面稳定
- [ ] 不要操作，让用户看清

#### 流畅书写（20 秒）

- [ ] 用鼠标流畅书写
- [ ] 写"Hello MindNotes"
- [ ] 展示笔迹平滑度
- [ ] 保持速度适中

#### 颜色切换（10 秒）

- [ ] 点击不同颜色
- [ ] 用不同颜色继续书写
- [ ] 展示颜色多样性

#### 粗细调整（10 秒）

- [ ] 点击粗细按钮
- [ ] 展示不同粗细效果
- [ ] 画几条不同粗细的线

#### 撤销功能（10 秒）

- [ ] 故意写错一个字
- [ ] 按 Ctrl+Z 撤销
- [ ] 展示撤销效果

#### 导出功能（20 秒）

- [ ] 点击保存按钮
- [ ] 选择 PNG 格式，展示效果
- [ ] 选择 PDF 格式，展示效果
- [ ] 选择 JSON 格式，展示效果

#### 清空画布（5 秒）

- [ ] 按 Delete 或点击清空
- [ ] 展示清空效果

#### 结尾展示（10 秒）

- [ ] 回到干净界面
- [ ] 展示 GitHub 链接
- [ ] 保持画面稳定

### 6. 录制检查

- [ ] 检查视频是否清晰
- [ ] 检查音频是否清晰（如有）
- [ ] 检查是否有卡顿
- [ ] 检查是否有误操作
- [ ] 检查时长是否合适（2-3 分钟）
- [ ] 如不满意，重新录制
