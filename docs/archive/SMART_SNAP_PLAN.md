# 📏 智能吸附功能实现计划

> 自动对齐辅助线和智能吸附

**优先级**: P1  
**预计时间**: 2 小时  
**状态**: 待实施

---

## 功能目标

实现类似 Figma、Excalidraw 的智能吸附功能，帮助用户绘制对齐的图形。

---

## 功能列表

### 1. 对齐辅助线

**水平/垂直对齐**:
- 检测形状之间的边缘对齐
- 显示对齐辅助线（黄色虚线）
- 自动吸附到对齐位置

**中心对齐**:
- 检测形状中心点对齐
- 显示中心辅助线
- 自动吸附到中心位置

---

### 2. 智能吸附

**边缘吸附**:
- 接近其他形状边缘时自动吸附
- 吸附阈值：5px
- 视觉反馈（辅助线高亮）

**等距分布**:
- 检测等距排列
- 显示等距辅助线
- 自动调整到等距位置

---

### 3. 网格吸附

**网格系统**:
- 可配置网格大小（默认 20px）
- 吸附到网格点
- 可选显示网格

---

## 技术实现

### 数据结构

```typescript
interface GuideLine {
  type: 'horizontal' | 'vertical'
  position: number
  alignType: 'start' | 'center' | 'end'
}

interface SnapPoint {
  x: number
  y: number
  type: 'edge' | 'center' | 'grid'
}
```

### 检测算法

1. **遍历所有形状**
2. **计算边缘位置** (start, center, end)
3. **检测距离** < 阈值
4. **生成辅助线**
5. **应用吸附**
