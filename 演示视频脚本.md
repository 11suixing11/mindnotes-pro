## 📹 录制指南

### 屏幕录制

**软件推荐**:
- OBS Studio（免费、专业）
- ScreenFlow（Mac）
- Camtasia（Windows/Mac）

**设置**:
```
分辨率：1920x1080 (1080p)
帧率：60fps
码率：10-20 Mbps
格式：MP4 (H.264)
```

**录制技巧**:
1. 清理桌面，关闭无关应用
2. 使用浏览器无痕模式
3. 调整浏览器窗口到合适大小
4. 鼠标移动要平滑
5. 操作速度适中（可后期加速）

### 旁白录制

**软件推荐**:
- Audacity（免费）
- Adobe Audition（专业）
- GarageBand（Mac）

**设置**:
```
采样率：48kHz
位深度：24-bit
格式：WAV（无损）
```

**录音技巧**:
1. 安静的环境（关闭空调、风扇）
2. 使用防喷罩
3. 距离麦克风 10-15cm
4. 保持语速适中（每分钟 150-180 字）
5. 保持微笑，声音会更温暖

## ✂️ 后期制作

### 剪辑软件

**推荐**:
- DaVinci Resolve（免费、专业）
- Adobe Premiere Pro（专业）
- Final Cut Pro（Mac）
- CapCut（简单易用）

### 剪辑要点

**节奏**:
- 开场：快速吸引（0.5-1 秒/镜头）
- 演示：适中展示（2-3 秒/镜头）
- 结尾：缓慢停留（3-5 秒）

**转场**:
- 场景切换：淡入淡出
- 重点强调：快速缩放
- 文字出现：打字机效果

**调色**:
- 亮度：适中（不要过曝）
- 对比度：轻微提升
- 饱和度：轻微提升
- 色温：偏暖（更友好）

### 字幕添加

**样式**:
```
字体：Noto Sans SC Bold
大小：24-32px
颜色：白色 + 黑色描边
背景：半透明黑色（可选）
位置：底部居中（安全区域）
```

**时机**:
- 与旁白同步
- 停留时间：2-4 秒
- 淡入淡出：0.2 秒
