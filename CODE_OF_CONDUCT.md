# Code of Conduct

MindNotes Pro should be a calm and useful place to collaborate.

## Expected Behavior

- Be respectful and specific when giving feedback.
- Assume mistakes are fixable and ask clarifying questions before escalating.
- Keep issue and PR discussions focused on the project.
- Credit prior work and upstream inspiration where relevant.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic comments after being asked to stop.
- Security disclosure details posted publicly before a fix is available.

## Enforcement

Maintainers may hide comments, close issues, block users, or limit participation when behavior makes collaboration unsafe or unproductive.

For conduct concerns, contact the maintainer through the public contact information on the GitHub profile. For security concerns, follow [SECURITY.md](SECURITY.md).
