# 🧩 MindNotes Pro 浏览器插件方案

> 让用户边浏览网页边做笔记

**创建时间**: 2026-03-19  
**状态**: 📝 方案规划

---

## 🎯 需求分析

### 用户场景

1. **看教程时** - 需要记录重点
2. **查资料时** - 需要整理信息
3. **看视频时** - 需要记笔记
4. **写论文时** - 需要引用网页

### 核心需求

- ✅ 不离开当前网页
- ✅ 快速打开笔记
- ✅ 随时记录想法
- ✅ 保存网页链接
- ✅ 截图标注

---

## 💡 实现方案

### 方案一：Chrome 浏览器插件 ⭐ 推荐

**优势**:
- ✅ 深度集成浏览器
- ✅ 快捷键触发
- ✅ 侧边栏显示
- ✅ 网页内容提取
- ✅ 截图标注

**实现**:
```
mindnotes-pro-extension/
├── manifest.json        # 插件配置
├── popup.html          # 弹出窗口
├── sidebar.html        # 侧边栏
├── content.js          # 网页内容脚本
├── background.js       # 后台脚本
└── styles.css          # 样式
```

**功能**:
1. 点击插件图标打开笔记
2. 快捷键 `Ctrl+Shift+M` 呼出
3. 侧边栏模式不遮挡网页
4. 自动保存当前网页 URL
5. 一键截图标注

---

### 方案二：悬浮球/磁铁模式

**优势**:
- ✅ 无需安装插件
- ✅ 所有浏览器通用
- ✅ 拖拽移动
- ✅ 始终置顶

**实现**:
```javascript
// 在现有项目中添加悬浮组件
FloatingNotes.tsx
- 可拖拽
- 可缩放
- 始终置顶
- 最小化到角落
```

**功能**:
1. 悬浮球点击展开笔记
2. 拖拽到任意位置
3. 调整大小
4. 透明度调节
5. 快捷键呼出

---

### 方案三：PWA 画中画模式

**优势**:
- ✅ 利用现有 PWA
- ✅ 无需额外开发
- ✅ 系统级支持

**实现**:
- 使用 Window Controls Overlay API
- Picture-in-Picture 模式
- 分屏使用

---

## 📦 技术选型

### 悬浮球方案

**依赖**:
- 现有 React + TypeScript
- 无需额外依赖
- 使用 framer-motion 动画

**核心 API**:
```javascript
// 拖拽
onMouseDown, onMouseMove, onMouseUp

// 位置存储
localStorage.setItem('floating-position')

// 快捷键
react-hotkeys-hook
```

---

### Chrome 插件方案

**依赖**:
- Chrome Extension API
- Manifest V3

**核心 API**:
```javascript
// 侧边栏
chrome.sidePanel.setOptions()

// 快捷键
chrome.commands.onCommand

// 存储
chrome.storage.local
```
