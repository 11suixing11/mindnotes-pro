## 💻 方式二：桌面应用（离线使用）⭐⭐⭐⭐

### 适用人群
- ✅ 需要离线使用
- ✅ 想要更好的性能
- ✅ 经常使用

### Windows 版

**步骤 1**: 下载安装包
```
下载：MindNotes Pro_1.0.0_x64_en-US.msi
```

**步骤 2**: 安装
```
1. 双击 .msi 文件
2. 点击"下一步"
3. 完成安装
```

**步骤 3**: 使用
```
双击桌面快捷方式 → 开始使用
```

### Mac 版

**步骤 1**: 下载 .dmg 文件
**步骤 2**: 拖到应用程序文件夹
**步骤 3**: 启动使用

### Linux 版

**步骤 1**: 下载 .AppImage 文件
**步骤 2**: 添加执行权限
```bash
chmod +x MindNotes-Pro.AppImage
```
**步骤 3**: 双击运行


## ❓ 常见问题

### Q1: 数据存在哪里？

**答**: 浏览器本地存储（所有方式都一样）

- 关闭应用不会丢失
- 可以导出备份
- 清除浏览器数据会删除

### Q2: 可以同步吗？

**答**: 目前不支持

每个设备的数据独立。可以导出 JSON 后在其他设备导入。

### Q3: 需要付费吗？

**答**: 完全免费！

- 在线版：免费
- 桌面应用：免费
- 移动端：免费
- 所有功能：免费

### Q4: 安全吗？

**答**: 非常安全！

- 数据存在本地
- 不上传云端
- 开源代码
- 无广告追踪

### Q5: 支持中文吗？

**答**: 完全支持！

- 中文界面
- 中文文档
- 中文支持

## 🎨 使用技巧

### 基础操作

| 操作 | 方法 |
|------|------|
| 绘写 | 鼠标/手指按住拖动 |
| 切换工具 | 点击工具栏 |
| 选择颜色 | 点击颜色圆点 |
| 调整粗细 | 点击粗细按钮 |
| 撤销 | Ctrl+Z / 点击撤销 |
| 清空 | Delete / 点击清空 |
| 保存 | Ctrl+S / 点击保存 |

### 高级技巧

1. **多颜色分类**
   - 用不同颜色区分内容
   - 红色：重点
   - 蓝色：普通
   - 绿色：完成

2. **快速导出**
   - PNG：快速分享
   - PDF：正式文档
   - JSON：备份/再编辑

3. **快捷键**
   ```
   Ctrl+Z    → 撤销
   Ctrl+S    → 保存
   Delete    → 清空
   Esc       → 关闭对话框
   ```
