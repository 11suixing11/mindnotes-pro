# Windows 用户使用指南

> 零门槛使用指南，小白也能轻松上手！

---

## 🎯 方式一：一键启动包（最简单）

### 步骤 1：下载项目

1. 访问：https://github.com/11suixing11/mindnotes-pro
2. 点击绿色的 **"Code"** 按钮
3. 选择 **"Download ZIP"**
4. 下载完成后解压

### 步骤 2：安装 Node.js（首次需要）

如果电脑上没有 Node.js：

1. 访问：https://nodejs.org/
2. 下载 **LTS 版本**（推荐 20.x）
3. 双击安装包，一直点"下一步"
4. 安装完成后**重启电脑**

### 步骤 3：启动应用

1. 打开解压的文件夹
2. **双击** `启动应用.bat`
3. 浏览器会自动打开
4. 开始使用！

**就这么简单！** 🎉

## 🎯 方式三：创建桌面快捷方式

### 创建快捷方式

1. 找到 `启动应用.bat` 文件
2. **右键** → **"发送到"** → **"桌面快捷方式"**
3. 桌面会出现快捷方式图标
4. 双击图标即可启动

### 更改图标（可选）

1. 右键快捷方式 → **属性**
2. 点击 **"更改图标"**
3. 选择一个喜欢的图标
4. 点击 **"确定"**

## ❓ 常见问题

### Q1: 双击没反应？

**原因**: 没有安装 Node.js

**解决**:
```
1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本
3. 重启电脑
4. 重新双击启动
```

---

### Q2: 提示"找不到 Node"？

**解决**:
```
1. 确认已安装 Node.js
2. 重启电脑
3. 重新双击启动
```

---

### Q3: 浏览器没有自动打开？

**解决**:
```
手动访问：http://localhost:3000
```

---

### Q4: 如何停止应用？

**方法 1**: 关闭黑色命令行窗口
**方法 2**: 按 `Ctrl+C`

---

### Q5: 数据存在哪里？

**答案**: 浏览器本地存储

- 关闭浏览器不会丢失
- 可以导出为文件备份
- 清除浏览器数据会删除

## 📸 使用截图

### 启动应用

```
1. 双击 启动应用.bat
2. 等待浏览器自动打开
3. 看到应用界面
```

### 开始绘写

```
1. 选择工具（笔/橡皮）
2. 选择颜色
3. 在白色区域按住鼠标绘写
4. 松开鼠标完成一笔
```

### 保存笔记

```
1. 点击右下角"保存"按钮
2. 选择格式（PNG/PDF/JSON）
3. 选择保存位置
4. 点击"保存"
```

## 🎨 使用技巧

### 基础操作

| 操作 | 方法 |
|------|------|
| 绘写 | 鼠标按住左键拖动 |
| 切换工具 | 点击工具栏按钮 |
| 选择颜色 | 点击颜色圆点 |
| 调整粗细 | 点击粗细按钮 |
| 撤销 | 按 `Ctrl+Z` |
| 清空 | 按 `Delete` |
| 保存 | 按 `Ctrl+S` |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+S` | 保存 |
