## 🎯 方式二：本地构建版

### 步骤 1：构建项目

```bash
cd mindnotes-pro
npm run build
```

### 步骤 2：使用本地服务器

**选项 A：使用 Python**

```bash
# Python 3
cd dist
python -m http.server 8000

# 访问：http://localhost:8000
```

**选项 B：使用 Node.js**

```bash
# 安装 serve
npm install -g serve

# 启动服务
serve dist

# 访问：http://localhost:3000
```

**选项 C：直接打开**

```
双击 dist/index.html
```

## 🎯 方式三：一键部署到自己的 Vercel

### 步骤

1. 访问：https://vercel.com/new
2. 点击 "Import Git Repository"
3. 选择 `mindnotes-pro` 仓库
4. 点击 "Deploy"
5. 等待 1-2 分钟
6. 获得你的专属网址

### 自定义域名（可选）

在 Vercel 设置中添加自己的域名。

## ❓ 常见问题

### Q1: 在线版数据存在哪里？

**答**：浏览器本地存储（IndexedDB）

- 关闭浏览器不会丢失
- 可以导出备份
- 清除浏览器数据会删除

### Q2: 可以在多个设备同步吗？

**答**：目前不支持

每个设备的数据独立存储。可以导出 JSON 后在其他设备导入。

### Q3: 需要联网吗？

**答**：首次访问需要

后续可以离线使用（PWA 支持开发中）。

### Q4: 支持哪些浏览器？

**答**：现代浏览器

- ✅ Chrome 90+
- ✅ Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ⚠️ IE 不支持
