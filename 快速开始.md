# 🚀 快速开始指南

> 5 分钟启动你的 MindNotes Pro！

---

## ⚡ 快速启动（3 步）

### 步骤 1：安装依赖

```bash
cd /home/admin/openclaw/workspace/mindnotes-pro
npm install
```

**预计时间**: 1-2 分钟

---

### 步骤 2：启动开发服务器

```bash
npm run dev
```

**预期输出**:
```
  VITE v5.1.6  ready in 500 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

浏览器会自动打开应用！

---

### 步骤 3：开始使用

1. **选择工具** - 点击工具栏的"笔"或"橡皮"
2. **选择颜色** - 点击颜色圆点
3. **选择粗细** - 点击粗细按钮
4. **开始绘写** - 在画布上按住鼠标左键拖动
5. **保存笔记** - 点击右下角"保存"按钮

## 🛠️ 常见问题

### Q1: 依赖安装失败

**错误**: `npm install` 报错

**解决**:
```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

---

### Q2: 端口被占用

**错误**: `Port 3000 is already in use`

**解决**:
```bash
# 方法 1: 使用其他端口
npm run dev -- --port 3001

# 方法 2: 关闭占用端口的程序
lsof -ti:3000 | xargs kill -9
```

---

### Q3: 浏览器没有自动打开

**解决**: 手动访问 http://localhost:3000

---

### Q4: 笔迹不流畅

**解决**:
1. 检查浏览器是否是最新版
2. 尝试使用 Chrome/Edge 浏览器
3. 关闭其他占用资源的程序
