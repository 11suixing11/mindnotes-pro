## 📲 安装为 PWA 应用

**推荐安装到主屏幕，像原生应用一样使用！**

### Chrome for Android

1. **打开网站**
   - 访问 https://11suixing11.github.io/mindnotes-pro

2. **打开菜单**
   - 点击右上角 ⋮ (三个点)

3. **添加到主屏幕**
   - 选择"添加到主屏幕"
   - 或"安装应用"

4. **确认安装**
   - 点击"添加"
   - 桌面会出现 MindNotes Pro 图标

5. **开始使用**
   - 像原生应用一样打开
   - 全屏显示
   - 离线可用

### Samsung Internet

1. 打开网站
2. 点击菜单 (≡)
3. 选择"添加页面到"
4. 选择"主屏幕"
5. 确认添加

### Firefox for Android

1. 打开网站
2. 点击菜单 (⋮)
3. 选择"页面"
4. 选择"添加到主屏幕"
5. 确认添加

## 🐛 常见问题

### Q1: 打开显示空白？

**解决**:
1. 检查网络连接
2. 清除浏览器缓存
3. 刷新页面
4. 尝试备用地址

### Q2: 无法添加到主屏幕？

**解决**:
1. 确保使用 Chrome 浏览器
2. 检查网站是否完全加载
3. 重启浏览器
4. 更新 Chrome 到最新版

### Q3: 离线无法使用？

**解决**:
1. 确保已安装 PWA
2. 至少打开过一次（有缓存）
3. 检查 Service Worker 状态
4. 重新安装 PWA

### Q4: 手写功能无法使用？

**解决**:
1. 确保使用最新版 Chrome
2. 检查触摸权限
3. 重启应用
4. 清除缓存
