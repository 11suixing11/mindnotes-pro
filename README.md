# MindNotes Pro

本地白板绘图应用 — React + Canvas + Zustand

## 功能

- 画笔 (平滑曲线)
- 橡皮擦除
- 平移画布
- 形状 (矩形、圆形)
- 8 种颜色 + 4 种线宽
- 撤销/重做
- 缩放 (滚轮/按钮)
- 网格背景
- 深色/浅色主题
- 键盘快捷键 (1-5 切换工具)

## 快捷键

| 按键 | 功能 |
|------|------|
| 1 | 画笔 |
| 2 | 橡皮 |
| 3 | 平移 |
| 4 | 矩形 |
| 5 | 圆形 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| + / - | 缩放 |
| Delete | 清空 |

## 开发

```bash
npm install

```

