```
┌─────────────────────────────────────────┐
│ ⚡ 输入命令或搜索功能...                 │

│   🗑️ 清空画布                          │
│                                         │
│ ➕ 插入                                  │
│   📝 插入文本框         T               │
│   🖼️ 插入图片          Ctrl+I          │
│   📋 使用模板                           │
│                                         │
│ 👁️ 视图                                  │
│   🎨 切换主题          Ctrl+Shift+L    │
│   🔍 放大/缩小        Ctrl+/-          │
│                                         │
│ 📤 导出                                  │
│   📤 导出为图片/PDF/Markdown            │
```

**特性**:
- ✅ 实时搜索过滤
- ✅ 键盘导航 (↑↓选择，Enter 执行)
- ✅ 类别分组显示
- ✅ 快捷键提示

### 📋 智能模板系统

**6 个专业模板，快速开始你的笔记！**

| 模板 | 用途 | 适用场景 |
|------|------|---------|
| 📝 **会议纪要** | 记录会议要点、决策和待办 | 团队会议、客户会议 |
| 📚 **学习笔记** | 康奈尔笔记法结构 | 课堂笔记、在线课程 |
| 💡 **头脑风暴** | 创意发想工具 | 策划方案、问题解决 |
| 📋 **待办事项** | 任务清单管理 | 每日计划、项目管理 |
| 🎯 **目标规划** | SMART 原则制定目标 | 年度目标、个人发展 |
| 📔 **每日笔记** | 晨间日记 + 晚间复盘 | 每日记录、反思成长 |

**使用方式**:
1. 按 `Ctrl+P` 打开命令面板
2. 输入"模板"
3. 选择适合的模板
4. 开始高效记录！

**效率提升**: 会议记录准备从 5 分钟缩短到**30 秒**！⚡
