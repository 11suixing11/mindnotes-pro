## 🏆 The Most Realistic Eraser Experience — Ever

<div align="center">
  <img src=".github/demo.gif" alt="MindNotes Pro Physics Eraser Demo" width="80%" />
</div>

> **Why is this different from every other whiteboard?**
>
> Every whiteboard has an eraser. **None have a physics engine.**
>
> ✨ **Pressure sensing** — Press hard to erase completely, light pressure for partial erasing
> ✨ **Wear simulation** — Your eraser gets duller with use, just like real life
> ✨ **Shape awareness** — Round, square, and chisel erasers with different behaviors
> ✨ **Tilt support** — Apple Pencil side-erasing, just like a real pencil
> ✨ **Particle effects** — Watch eraser dust fly as you erase
> ✨ **Audio feedback** — Hear the scratch of the eraser, volume changes with pressure
> ✨ **3 eraser presets** — 2B hard, 4B medium, 6B soft art erasers

This isn't just "delete strokes." This is **erasing**.

## Keyboard shortcuts

Press `?` in the app to view all shortcuts. The essentials:

| Shortcut              | Action                            |
| --------------------- | --------------------------------- |
| `P`                   | Pen tool                          |
| `E`                   | Eraser (physics mode)             |
| `S`                   | Selection tool                    |
| `T`                   | Text tool                         |
| `R` / `C` / `L` / `A` | Rectangle / Circle / Line / Arrow |
| `Space` + drag        | Pan canvas                        |
| `Ctrl` + `Z` / `Y`    | Undo / Redo                       |
| `Ctrl` + `A`          | Select all                        |
| `Ctrl` + `C` / `V`    | Copy / Paste                      |
| `Ctrl` + `E`          | Export menu                       |
| `Delete`              | Delete selected                   |
| `Scroll`              | Zoom in/out                       |
| `Dark/Light`          | Toggle theme                      |

**Eraser-specific shortcuts:**

| Shortcut              | Action                            |
| --------------------- | --------------------------------- |
| `1` / `2` / `3`       | Switch eraser shape               |
| `Q` / `W` / `E`       | Switch eraser preset (2B/4B/6B)   |
| `R`                   | Sharpen eraser (reset wear)       |
| `M`                   | Toggle eraser sound               |
| `[` / `]`             | Adjust eraser size                |

### Just open it

👉 **[Open MindNotes Pro](https://11suixing11.github.io/mindnotes-pro)** — nothing to install, nothing to sign up for. **Try the physics eraser!**

### Run locally

```bash
git clone https://github.com/11suixing11/mindnotes-pro.git
cd mindnotes-pro
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — start drawing in 30 seconds.

## Tech stack

| Layer   | Choice                                | Why                           |
| ------- | ------------------------------------- | ----------------------------- |
| UI      | React 18 + TypeScript                 | Type-safe, fast               |
| State   | Zustand (6 slices)                    | Tiny, no boilerplate          |
| Drawing | perfect-freehand + Canvas API         | Natural pen strokes           |
| **Eraser** | **Custom Physics Engine + RBush** | **Industry exclusive**        |
| Style   | Tailwind CSS + Monet palette          | Beautiful by default          |
| Export  | jsPDF (lazy-loaded)                   | Zero impact on initial bundle |
| Build   | Vite 5                                | Instant HMR                   |
| Test    | Vitest + Testing Library              | 574+ unit tests               |
