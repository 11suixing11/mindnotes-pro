### 核心算法

**笔迹平滑** (perfect-freehand):
```typescript
const pathData = getStroke(points, {
  size: 4,           // 笔迹大小
  thinning: 0.5,     // 压感敏感度
  smoothing: 0.5,    // 平滑度
  streamline: 0.5,   // 防抖动
})
```

**状态管理** (Zustand):
```typescript
interface AppState {
  strokes: Stroke[]      // 完成的笔迹
  currentStroke: Stroke  // 当前笔迹
  tool: 'pen' | 'eraser' // 工具
  color: string          // 颜色
  size: number           // 粗细
  // ... 方法
}
```

## 🚀 如何使用

### 开发模式

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问 http://localhost:3000
```

### 生产构建

```bash
# 构建
npm run build

# 预览
npm run preview
```

### 部署

**推荐平台**: Vercel / Netlify

```bash
# 构建
npm run build

# 输出目录：dist/
# 上传到任意静态托管平台
```
