# Roadmap

This roadmap is intentionally small. MindNotes Pro should stay understandable and local-first.

## Current Priorities

- Stabilize canvas selection, keyboard shortcuts, undo/redo, and export behavior.
- Improve test coverage around history, snapping, alignment, SVG import/export, and persistence.
- Document design tradeoffs and known limitations more clearly.
- Keep the app usable offline and avoid hidden data upload.

## Later

- Better accessibility for toolbar, menus, dialogs, and keyboard-only workflows.
- More robust import/export edge cases.
- Smaller internal modules where large canvas handlers have become hard to review.
- Optional desktop packaging once the web app behavior is stable.

## Non-goals

- Real-time multiplayer collaboration.
- Account systems or hosted cloud storage.
- Analytics that track drawing content or user documents.
