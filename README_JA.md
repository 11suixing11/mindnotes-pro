## なぜ MindNotes Pro なのか

<table>
<tr>
<td width="50%">

### 🎨 ツールではなく、アートのような体験

**モネの印象派パレット**にインスパイア — 水彩グラデーション、グラスモーフィズム、紙のテクスチャで、デジタル描画を温かく人間らしいものに。カリグラフィー、グロー、鉛筆など**6種の筆ブラシ**。

### 🔒 あなたの考えはあなたのもの

**すべてブラウザ内に保存。** アカウント不要、サーバーなし、「ログインして保存」もなし。データはデバイスから外に出ない — PWA でオフラインでも動作。

### ⚡ ゼロ・バloat、瞬間起動

ランタイム依存は**たった3つ**。1秒以内にロード。50MBのElectronアプリではない — 常に準備完了のホワイトボード。

</td>
<td width="50%">

### 📝 視覚的に考える

- ✏️ フリーハンド描画、**6種のブラシスタイル**
- 🔷 図形 — 四角、円、線、矢印
- 📝 インラインテキスト注釈
- 🖼️ 画像をキャンバスに直接ペースト
- 🖱️ フレーム選択、リサイズ、移動、スナップ＆整列
- ↩️ 完全なアンドゥ/リドゥ履歴
- 🌙 ダークモード（システム設定を自動検出）
- 📄 **PDF** または **PNG** にエクスポート

### 🗂️ きちんと整理

- マルチドキュメントワークスペース
- フォルダ階層のドラッグ＆ドロップ
- localStorage に自動保存
- リ名前、複製、削除 — すべてローカルで

</td>
</tr>
</table>

## キーボードショートカット

アプリ内で `?` を押して全ショートカットを表示。基本：

| ショートカット        | アクション            |
| --------------------- | --------------------- |
| `P`                   | ペンツール            |
| `E`                   | 消しゴム              |
| `S`                   | 選択ツール            |
| `T`                   | テキストツール        |
| `R` / `C` / `L` / `A` | 四角 / 円 / 線 / 矢印 |
| `Space` + ドラッグ    | キャンバス移動        |
| `Ctrl` + `Z` / `Y`    | アンドゥ / リドゥ     |
| `Ctrl` + `A`          | すべて選択            |
| `Ctrl` + `C` / `V`    | コピー / ペースト     |
| `Ctrl` + `E`          | エクスポートメニュー  |
| `Delete`              | 選択項目を削除        |
| `スクロール`          | ズームイン/アウト     |
| `Dark/Light`          | テーマ切替            |

👉 **[MindNotes Pro を開く](https://11suixing11.github.io/mindnotes-pro)** — インストール不要、アカウント不要。

### ローカルで実行

```bash
git clone https://github.com/11suixing11/mindnotes-pro.git
cd mindnotes-pro
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開く — 30秒で描画開始。

## 技術スタック

| レイヤー     | 選定                                  | 理由                        |
| ------------ | ------------------------------------- | --------------------------- |
| UI           | React 18 + TypeScript                 | 型安全、高速                |
| 状態管理     | Zustand（6スライス）                  | 軽量、ボイラープレートなし  |
| 描画         | perfect-freehand + Canvas API         | 自然な筆跡                  |
| スタイリング | Tailwind CSS + モネパレット           | デフォルトで美しい          |
| エクスポート | jsPDF（遅延ロード）                   | 初期バンドルに影響なし      |
| ビルド       | Vite 5                                | 即時HMR                     |
| テスト       | Vitest + Testing Library + Playwright | 245+ユニットテスト、E2E対応 |
