## 🏆 史上最もリアルな消しゴム体験

<div align="center">
  <img src=".github/demo.gif" alt="MindNotes Pro 物理消しゴムデモ" width="80%" />
</div>

> **なぜ他のすべてのホワイトボードと違うのか？**
>
> どのホワイトボードにも消しゴムはある。**だが物理エンジンを搭載しているのはこれだけ。**
>
> ✨ **圧力感知** — 強く押すと完全に消去、弱く押すと薄くなる
> ✨ **摩耗シミュレーション** — 使うほど消しゴムが鈍くなる、まるで本物
> ✨ **形状認識** — 丸型、角型、チゼル型、それぞれ異なる消え方
> ✨ **傾きサポート** — Apple Pencil の斜め消し、本物の鉛筆のように
> ✨ **パーティクル効果** — 消しカスが飛び散る様子をリアルに再現
> ✨ **オーディオフィードバック** — 消す音が圧力で変化
> ✨ **3種の消しゴムプリセット** — 2B硬質、4B標準、6B軟質

これは単なる「ストロークの削除」ではない。これは**本当の消去**だ。

### 🎯 物理消しゴムエンジン — 業界独占

**この機能1つに22ラウンドの最適化を投入：**

- **圧力感知** — 強く押すと完全消去、弱く押すとストロークが薄くなる
- **摩耗シミュレーション** — 消しゴムが鈍くなる、鉛筆を削るように「消しゴムを削る」

**すべてブラウザ内に保存。** アカウント不要、サーバーなし、「ログインして保存」もなし。データはデバイスから外に出ない — PWA でオフラインでも動作。

### ⚡ ゼロ・バloat、瞬間起動

ランタイム依存は**たった5つ**。1秒以内にロード。50MBのElectronアプリではない — 常に準備完了のホワイトボード。

## キーボードショートカット

アプリ内で `?` を押して全ショートカットを表示。基本：

| ショートカット        | アクション            |
| --------------------- | --------------------- |
| `P`                   | ペンツール            |
| `E`                   | 消しゴム（物理モード） |
| `S`                   | 選択ツール            |
| `T`                   | テキストツール        |
| `R` / `C` / `L` / `A` | 四角 / 円 / 線 / 矢印 |
| `Space` + ドラッグ    | キャンバス移動        |
| `Ctrl` + `Z` / `Y`    | アンドゥ / リドゥ     |
| `Ctrl` + `A`          | すべて選択            |
| `Ctrl` + `C` / `V`    | コピー / ペースト     |
| `Ctrl` + `E`          | エクスポートメニュー  |
| `Delete`              | 選択項目を削除        |
| `スクロール`          | ズームイン/アウト     |
| `Dark/Light`          | テーマ切替            |

**消しゴム専用ショートカット：**

| ショートカット        | アクション                  |
| --------------------- | --------------------------- |
| `1` / `2` / `3`       | 消しゴム形状切替            |
| `Q` / `W` / `E`       | 消しゴムプリセット切替（2B/4B/6B） |
| `R`                   | 消しゴムを削る（摩耗リセット） |
| `M`                   | 消去音のオン/オフ          |
| `[` / `]`             | 消しゴムサイズ調整          |

## 技術スタック

| レイヤー     | 選定                                  | 理由                        |
| ------------ | ------------------------------------- | --------------------------- |
| UI           | React 18 + TypeScript                 | 型安全、高速                |
| 状態管理     | Zustand（6スライス）                  | 軽量、ボイラープレートなし  |
| 描画         | perfect-freehand + Canvas API         | 自然な筆跡                  |
| **消去**     | **独自開発物理エンジン + RBush** | **業界独占**        |
| スタイリング | Tailwind CSS + モネパレット           | デフォルトで美しい          |
| エクスポート | jsPDF（遅延ロード）                   | 初期バンドルに影響なし      |
| ビルド       | Vite 5                                | 即時HMR                     |
| テスト       | Vitest + Testing Library              | 574+ユニットテスト          |
