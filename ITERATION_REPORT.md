## 🔧 技术实现

### 核心算法
```typescript
// src/canvas/canvasUtils.ts
export function isTransparentImagePixel(
  imageEl: ImageElement,
  canvasX: number,
  canvasY: number
): boolean
```

### 实现细节
1. **离屏 Canvas 缓存**
   - 创建 1x1 离屏 canvas 检测像素
   - 使用 LRU 缓存策略，最大缓存 50 张图片
   - 缓存 key: imageElement.id + 图片尺寸

2. **透明度阈值**
   - alpha < 32（约12%不透明度）视为透明
   - 允许点击穿透到下方元素

3. **集成位置**
   - `src/components/canvas/usePointerEngine.ts`
   - hitTest() 函数中空间索引候选集路径
   - hitTest() 函数中降级遍历路径
